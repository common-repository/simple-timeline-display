== Installation ==

Download " simple-timeline-display.zip" to your computer.
Login wordpress as administrator
Click on Plugins --> Add New
On "Install Plugins" page, clik on "Upload"
Browse and select " simple-timeline-display.zip" and click on "Install Now"

You can activate it now

= Just use the following Shortcode  to you page or post =
[bs_timeline text=”simple timeline display(,) simple timeline display”]
